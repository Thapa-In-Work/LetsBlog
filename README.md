# LetsBlog

A responsive blog template with 7 blog post pages, built with HTML, CSS, and Bootstrap.

## Features

- Responsive design (desktop, tablet, mobile)
- Bootstrap carousel hero section
- Navigation with dropdown menus
- Mobile side menu
- Blog post pages with "See More" recommendations
- Post navigation (prev/next)
- Comment forms on each post
- Newsletter subscription widgets
- Social media links

## Pages

- **Home** (`index.html`) - Main landing page with carousel and blog grid
- **BEAUTY OF NATURE** (`pages/blog1.html`)
- **FASHION MODEL SHOOT** (`pages/blog2.html`)
- **GOLDEN SNOW LAND** (`pages/blog3.html`)
- **URBAN EXPLORATION** (`pages/blog4.html`)
- **EXTREME SPORTS** (`pages/blog5.html`)
- **WILDERNESS CAMP** (`pages/blog6.html`)
- **MORNING RITUALS** (`pages/blog7.html`)

## Usage

Open `index.html` in any modern web browser. No build tools or server required.

## GitHub Pages

This site is ready for GitHub Pages. Simply push to a `main` branch and enable Pages in the repository settings.
